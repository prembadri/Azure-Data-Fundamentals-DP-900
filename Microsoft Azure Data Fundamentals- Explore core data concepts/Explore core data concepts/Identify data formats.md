# Identify data formats
Data is a collection of facts such as numbers, descriptions, and observations used to record informations.

 1. Data structures in which this data is organized often represents **_entities_** that are important to an organization (such as customers, products, sales orders, and so on).
 2. Each entity typically has one or more **_attributes_**, or characteristics (for example, a customer might have a name, an address, a phone number, and so on)

There are 3 types of data in the real world 

 1. Structured Data - TABLES are best example
 2. Semi-Structured Data - JSON, XML, EXCEL etc., are best examples
 3. Un Structured Data - DOCUMENTS, IMAGES, AUDIO & VIDEO, BINARY FILES are best examples.

## Data Stores 
Organizations typically store data in structured, semi-structured, or unstructured format to record details of ***entities***, ***specific events*** or other information in documents, images, and other formats. The stored data can then be retrieved for analysis and reporting later. 

**entities**: for example, customers and products
**specific events:** sales transactions 

There are two broad categories of data store in common use:
-   File stores
-   Databases
